# Dubey - Per-App Time Limiter

Personal parental control Android app for Realme 9 5G.

## Features
- Per-app daily time limits
- Beautiful Hindi blocking overlay
- PIN-protected settings
- Secret extra time (tap 10 times)
- Survives ColorOS aggressive battery optimization

## Build
Push to GitHub and download APK from Actions artifacts.

## Setup on Device
1. Install APK
2. Grant Usage Access permission
3. Grant Display Over Other Apps permission
4. Disable Battery Optimization
5. Enable Auto-start
6. Lock in recent apps

## Default PIN
1234 (change after first login)

## Milestone 1 Testing
- Hardcoded limits: YouTube (10 min), Instagram (10 min)
- Test for 24 hours on actual device
